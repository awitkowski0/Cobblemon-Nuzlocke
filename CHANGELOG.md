# Changelog

## 1.0.0

### Features
- **Auto-Release** - Dead Cobblémon are automatically released from your party
- **In-Game Config** - Admins can configure rules via commands
- **JSON Config** - All settings saved to `config/nuzlocke/nuzlocke.json`

### Commands
- `/nuzlocke start` - Start a new run
- `/nuzlocke stop` - End current run
- `/nuzlocke status` - View run stats
- `/nuzlocke graveyard` - View fallen Cobblémon
- `/nuzlocke rules` - View current rules
- `/nuzlocke config <option>` - Admin configuration
